API Rest - Comunas y Vecinos

  • Ids de las Comunas

Comunas Endpoints

GET /api/comunas

Obtener todas las comunas

POST /api/comunas

Crear una nueva comuna

Enviar un JSON de esta forma:

      {
        "comuna": 16,
        "nombre": "Comuna 16",
        "descripcion": "Descripción de la comuna 16",
        "dato_adicional": "Dato adicional de la comuna 16",
        "comisarias": {
          "comunal": {
            "nombre": "Comisaría Comunal 16",
            "direccion": "Dirección de la Comisaría Comunal 16"
          },
          "vecinales": [
            {
              "nombre": "Comisaría Vecinal 16-A",
              "direccion": "Dirección de la Comisaría Vecinal 16-A"
            },
            {
              "nombre": "Comisaría Vecinal 16-B",
              "direccion": "Dirección de la Comisaría Vecinal 16-B"
            }
          ]
        },
        "barrios": ["Barrio 1", "Barrio 2", "Barrio 3"]
      }
PUT /api/comunas/:id

Actualizar una comuna existente (Reemplazar :id con el ID real)

Enviar un JSON de esta forma:

      {
        "comuna": 100,
        "nombre": "Comuna actualizada",
        "descripcion": "Descripción actualizada de la comuna",
        "dato_adicional": "Nuevo dato adicional",
        "comisarias": {
          "comunal": {
            "nombre": "Comisaría Comunal actualizada",
            "direccion": "Av. Comunal 123"
          },
          "vecinales": [
            {
              "nombre": "Comisaría Vecinal actualizada",
              "direccion": "Calle Vecinal 456"
            }
          ]
        },
        "barrios": ["Barrio A", "Barrio B", "Barrio C"]
      }
DELETE /api/comunas/:id

Eliminar una comuna existente (Reemplazar :id con el ID real)

Vecinos Endpoints

GET /api/vecinos

Obtener todos los vecinos

GET /api/vecinos/:id

Buscar un vecino por su ID (Reemplazar :id con el ID real)

GET /api/vecinos/nombre/:nombre

Buscar un vecino por su nombre (Reemplazar :nombre con el nombre exacto. No distingue mayúsculas/minúsculas).

Por ejjemplo:

     /api/vecinos/nombre/Lionel Messi
GET /api/vecinos/comuna/:comunaId

Obtener todos los vecinos de una comuna (Reemplazar :comunaId con el ID real de la comuna).

Por ejemplo:

     /api/vecinos/comuna/680fb52f5c1cd6336914d11b
POST /api/vecinos

Crear un nuevo vecino

Enviar un JSON de esta forma:

      {
        "nombre": "Lionel Messi",
        "edad": 36,
        "comuna": "680fb52f5c1cd6336914d11b",
        "direccion": "Calle de la Copa 123",
        "telefono": "123456789"
      }
PUT /api/vecinos/:id

Actualizar un vecino existente (Reemplazar :id con el ID real)

Enviar un JSON de esta forma:

{
  "nombre": "Lionel Andres Messi",
  "direccion": "Calle Actualizada 123",
  "telefono": "9876-5432",
  "comunaId": "680fb52f5c1cd6336914d11c"
}
DELETE /api/vecinos/:id

Eliminar un vecino existente (Reemplazar :id con el ID real)
